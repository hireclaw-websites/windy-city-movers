# Windy City Movers

Chicago's Trusted Moving Company Website

## Overview

A modern, responsive website for Windy City Movers - a professional moving company serving Chicago since 1995.

## Features

- **Responsive Design** - Works on all devices
- **Quote Calculator** - Instant moving estimates
- **Contact Form** - Easy customer inquiries
- **Smooth Animations** - Enhanced user experience
- **Chicago Theme** - Inspired by the Chicago flag colors

## Live Site

🌐 **https://windy-city-movers.pages.dev**

## Technologies

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Cloudflare Pages

## Local Development

```bash
# Clone the repository
git clone https://github.com/hireclaw-websites/windy-city-movers.git

# Navigate to project
cd windy-city-movers

# Open in browser
open index.html
```

## Deployment

This site is automatically deployed via Cloudflare Pages on every push to the `main` branch.
